## Installation

1. Create a .env file

```bash
DATABASE_URL="postgresql://postgres:123456@localhost:5432/nameDB?schema=public"
```

2. Run user seeds

```bash
npm run seed:user
```

3. Run codes seeds

```bash
npm run seed:code
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
